§Introduction

Rust bindings for the igraph C library for creating and manipulating graphs.

We are using the bindgen crate to build Rust bindings over the igraph C library, after the publication of Csardi, G., & Nepusz, T. (2006). The igraph software package for complex network research. InterJournal, Complex Systems, 1695.

The latest version of the original C library can be fetched on Github and the installation proceeds according to the tutorial:

wget https://github.com/igraph/igraph/releases/download/1.0.0/igraph-1.0.0.tar.gz
tar xf igraph-1.0.0.tar.gz
cd igraph-1.0.0
mkdir build && cd build
cmake .. # -DBUILD_SHARED_LIBS=ON for building shared libraries.
cmake --build .
sudo cmake --install .

provided that the main tarball as been extracted and that the following dependencies

brew install gmp lapack libxml2 glpk arpack blas suitesparse

are installed as well.
